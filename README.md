Emoji-Web
---------

## Requirements

  - Python 3.5

## Getting Started

```
$ python --version
Python 3.5.0

$ pip install -r requirements.txt
$ python app.py
```

### venv

It is recommended that you use the venv.

If in *inux environemnts:

```
$ python -m venv venv
$ source venv/bin/activate

...

$ deactivate # exit
```
